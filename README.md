# student-portal-website
IMS566 ADVANCED WEB DESIGN DEVELOPMENT AND CONTENT MANAGEMENT

## Overview
This is a student portal application that allows students to log in and access various features including:
- Dashboard to view an overview of the current semester.
- Courses to view registered courses and their details.
- Results to check current exam results.
- Activity to view photos and videos from events and classes.
- Profile to view personal information.

## Features
- Login: Secure login with username and password.
- Dashboard: Overview of the student's current courses, CGPA and credits for the semester.
- Courses: List of registered courses, credit hours and lecturers.
- Results: Current exam results with grade breakdown.
- Activity Gallery: View photos and videos from events and classes.
- Profile: View personal details and social media links.

## File Structure
- index.html: Login page for user authentication.
- dashboard.html: Dashboard displaying semester overview and progress.
- courses.html: Page showing registered courses and their details.
- results.html: Page displaying exam results.
- activity.html: Gallery of photos and videos from events.
- profile.html: Personal profile page with details and social links.
- style.css: Stylesheet defining the layout and design of the portal.

## Technologies Used
- HTML: Structure of the web pages.
- CSS: Styling for layout, color schemes, and responsiveness.
- JavaScript: For form validation, dynamic content (e.g., GPA chart), and logout functionality.
- Chart.js: Used for displaying GPA progress as bar and line charts.

## Setup and Installation
1. Clone this repository to your local machine.
2. Open the HTML files in a web browser to explore the portal.
